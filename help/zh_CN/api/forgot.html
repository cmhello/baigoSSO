公共请求参数

公共请求参数是指向所有接口发起请求时都必须传入的参数。

名称 类型 必须 描述
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 应用。
time int true Unix 时间戳，baigo SSO 允许 +-30 分钟以内的时差，为了防止时区设置不同导致的时差，请开发者将应用的时区设置为为与 baigo SSO 一致，关于时区设置，请查看 系统设置。
sign string true 签名，将请求参数中的加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，加以签名，签名方式请查看 签名。

通过邮件找回

接口说明

本接口用于已注册用户找回密码，需要通过邮件进行验证。

URL

http://www.domain.com/api/api.php?m=forgot&c=api

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

否

 

请求参数

名称 类型 必须 描述
a string true 接口调用动作，值只能为 bymail。
code string true 加密参数，将加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。
 

加密参数

将下列参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。

名称 类型 必须 描述
user_id int true

三选一，优先级为 user_id > user_name > user_mail

其中是否允许邮箱登录，视注册设置情况而定。详情查看 注册设置。
user_name string
user_mail string
 

返回结果

名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
prd_sso_ver string baigo SSO 版本号。
prd_sso_pub string baigo SSO 版本发布时间，格式为年月日。
 
返回结果示例

{
    "rcode": "y010408"
}

回答密保问题找回

此接口自 v2.0 开始生效

接口说明

本接口用于已注册用户找回密码，需要回答密保问题直接修改密码。

URL

http://www.domain.com/api/api.php?m=forgot&c=api

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

否

 

请求参数

名称 类型 必须 描述
a string true 接口调用动作，值只能为 byqa。
code string true 加密参数，将加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。
 

加密参数

将下列参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。

名称 类型 必须 描述
user_id int true

三选一，优先级为 user_id > user_name > user_mail

其中是否允许邮箱登录，视注册设置情况而定。详情查看 注册设置。
user_name string
user_mail string
user_sec_answ_1 string true 密保答案 1
user_sec_answ_2 string true 密保答案 2
user_sec_answ_3 string true 密保答案 3
 

返回结果

名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
prd_sso_ver string baigo SSO 版本号。
prd_sso_pub string baigo SSO 版本发布时间，格式为年月日。
 
返回结果示例

{
    "rcode": "y010407" //返回代码
}
