生成签名

自 v3.0 开始取消签名接口，请开发者自行按照加解密规则编写加解密程序；./_example/sign.class.php 文件为签名示例，开发者可以此为例自行编写程序。

 
生成签名函数示例

function sign_make($string, $app_key, $app_secret) {
	if (get_magic_quotes_gpc()){
    	$string = stripslashes($string);
	}

	return strtoupper(md5($app_key . $string . $app_secret));
}

 

参数说明

名称 类型 必须 描述
string string true 待签名字符串
app_key string true APP KEY 通信密钥，可从后台应用管理界面获得。
app_secret string true APP KEY 密文密钥，长度必须为 16 位, 可从后台应用管理界面获得，此参数为加解密使用，请注意保密，切勿在网络中传输。
