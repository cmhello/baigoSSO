用户注册

接口说明

本接口用于新用户的注册。用户名、密码、邮箱、昵称为一个用户在 baigo SSO 的基本数据，提交后 baigo SSO 会按照 注册设置 检测用户名和邮箱的格式是否正确合法。

URL

http://www.domain.com/api/api.php?m=user&c=api

HTTP 请求方式

POST

返回格式

JSON

返回结果是否加密

是

 

请求参数

名称 类型 必须 描述
a string true 接口调用动作，值只能为 reg。
code string true 加密参数，将加密参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。
 

加密参数

将下列参数的值用 Base64 编码（注：切勿将参数名进行 Base64 编码），然后拼合成 JSON 对象，最后用加密函数加密，加密方法请查看 密文。

名称 类型 必须 描述
user_name string true 用户名
user_pass string true 密码，必须用 MD5 加密后传输。
user_mail string 视注册设置情况而定。详情查看 注册设置。
user_nick string false 昵称
user_contact string false 联系方式，详情查看 联系方式请求格式。
user_extend string false 扩展字段，详情查看 扩展字段请求格式。
user_ip string false 用户当前 IP 地址
 

返回结果

本接口返回加密结果，真正内容需要解密。详情查看 密文。

名称 类型 描述
rcode string 返回代码，详情查看 返回代码。
code string 加密结果，需要解密。详情查看 密文。
sign string 签名，即本接口解密后的结果（JSON 对象，不进行 JSON 和 Base64 解码）加以签名。签名验证方式请查看 签名。
prd_sso_ver string baigo SSO 版本号。
prd_sso_pub string baigo SSO 版本发布时间，格式为年月日。
 

解密后的结果

名称 Base64 类型 描述
user_id true int 用户 ID
 
解密结果示例

{
    "user_id": "MTA=" //用户 ID
}
