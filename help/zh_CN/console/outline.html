左侧菜单组成说明
菜单 功能 说明
用户管理 用户管理功能 已注册的用户管理，您可以在此对用户进行各种管理操作。
应用管理 应用管理功能 要整合 baigo SSO，必须在此创建应用，并对应用授予相应的权限，这些应用通过 API 接口调用来实现各种功能。
日志管理 日志管理功能 后台、应用的部分操作，会以日志的形式记录在此，方便查看和调试。
管理员 管理员管理功能  
系统设置 系统设置功能 您可以在此设置网站名称、时区、各类格式等。
